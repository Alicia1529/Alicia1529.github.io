Tencent Cloud

  • Use account and password to login

  1. Connect to the server ssh <username>@<hostname or IP address>

  2. Enter password

  • Use key to login

  1. Download the key file chmod 400 <下载的与云服务器关联的私钥的绝对路径>

  2. Remote login ssh -i <下载的与云服务器关联的私钥的绝对路径> <username>@<hostname or IP address>
    eg. ssh -i "Mac/Downloads/shawn_qcloud_stable" ubuntu@192.168.11.123

  3. Change authrization: chmod 400 <file> to enable copy operation

  4. Notes: SSH远程登录的步骤与技巧
