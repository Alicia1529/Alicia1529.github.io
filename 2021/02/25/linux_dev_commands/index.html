Docker 命令

  • 根据dockerfile在当前目录创建镜像: docker build . -t <repo>:<tag>

  • 删除镜像or容器: docker rm <id>

  • 运行镜像: docker run -it -v <local dir>:<remote dir> <repo>:<tag> /bin/bash

  • docker 删除缓存 docker system prune --volumes

  • 容器改名 docker rename <container_id> <new name>

  • 保存镜像 docker commit [OPTIONS] CONTAINER [REPOSITORY[:TAG]]
